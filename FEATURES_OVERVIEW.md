## 🔬 Experiment Configuration

### Exposure Time Options
- ⏱️ **0.1 seconds** - Extreme speed, maximum difficulty
- ⏱️ **0.3 seconds** - Quick presentation (recommended for practice)
- ⏱️ **0.5 seconds** - Moderate speed
- ⏱️ **1.0 seconds** - Extended viewing, easiest condition

### Number of Trials
- 🔁 **Configurable Range**: 3-20 trials per experiment
- 🎯 **Default**: 10 trials (optimal balance)
- ⚙️ **Flexible**: Adjust based on class time or individual needs

### Stimulus Types

#### Letters
- Classic letter stimuli (A-Z)
- Familiar symbols, easy recognition
- Tests basic attention

#### Numbers  
- Digit stimuli (0-9)
- Familiar numerical symbols
- Similar difficulty to letters

#### Shapes
- Visual geometric shapes (▲ ● ■ ★ ◆ ▼ ○)
- More complex encoding needed
- Tests spatial attention

#### Words
- Common English vocabulary
- CAT, DOG, SUN, MOON, TREE, etc.
- Tests semantic attention

#### Mixed
- Combination of all types
- Maximum difficulty
- Tests flexible attention

### Stimulus Variations
- 🔀 **Random Generation**: Each trial has unique stimulus
- 📊 **Configurable Count**: 3-10 items per trial
- 🎲 **Randomized Order**: Prevents predictability
- 📈 **Progressive Difficulty**: Gradual increase (optional)
